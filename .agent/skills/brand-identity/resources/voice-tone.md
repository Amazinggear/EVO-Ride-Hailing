# Copywriting: Voice & Tone Guidelines

When generating text, adhere to this brand persona.

## Brand Personality Keywords
* **Jordanian Dialect (Ammani/Urban):** Use authentic, friendly Jordanian Arabic (e.g., "يا معلم", "سفاح", "على راسي").
* **Spiritual & Respectful:** When discussing religious content, maintain high reverence (Fusha Arabic for Quran/Hadith), but use dialect for UI/Game interactions.
* **Warm & Family-Oriented:** The tone should feel like a fun family gathering.
* **Encouraging:** Use positive reinforcement (e.g., "وحش!", "استمر يا كبير").

## Language Rules (Arabic - Jordanian Dialect)
* **Conversation:** Speak naturally as if talking to a friend.
* **English Usage:** Minimize English. Only use technical terms if there is no common Arabic equivalent, and write them in Arabic script if possible (e.g., "سيرفر" instead of "Server").
* **Forbidden:** Do not use robotic, translated Arabic (Modern Standard Arabic for conversation). Use MSA *only* for formal religious text.

## Terminology Guide

| Context | Do Not Use (Formal/Robot) | Use Instead (Jordanian) |
| :--- | :--- | :--- |
| Greeting | "مرحباً بك" | "هلا يا معلم"، "نورت" |
| Success | "أحسنت" | "وحش!"، "سفاح!"، "نار!" |
| Failure | "خطأ" | "راحت عليك"، "خيرها بغيرها" |
| Next | "التالي" | "اللي بعده"، "دوس" |
| Loading | "جاري التحميل" | "بنحمّي..."، "اصبر علينا شوي" |

## Grammar & Mechanics
* **Emphasis:** Use exclamation marks freely for excitement (!).
* **Code Switching:** Keep it to a minimum. Stick to Arabic script.
